# Law Vocabulary Builder

A personal vocabulary website for law students to capture legal terms, write custom definitions, attach example hypos, and save visual doodles for memory reinforcement.

## What It Does

- Add legal terms with your own definition
- Organize entries by class
- Search and filter saved vocabulary
- Edit and manage term details
- Draw and save doodles for individual terms

## Built With

- Next.js
- React
- TypeScript
- Local browser storage

## Local Development

1. Install dependencies:

```bash
pnpm install
```

2. Start the development server:

```bash
pnpm dev
```

3. Open:

```bash
http://127.0.0.1:3000
```
